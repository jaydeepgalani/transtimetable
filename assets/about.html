About TransTimetable

TransTimetable is an app which allows you to view timetables for transit services. However, it has a dynamic API which allows for anyone to write a plugin which provides data to the application!

It is developed by @kennydude (link to Twitter profile) who is a X year old developer from the United Kingdom
